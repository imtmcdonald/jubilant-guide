# Security Policy

## Reporting a vulnerability

Please do not open a public issue for security-sensitive reports.

Instead, use GitHub Security Advisories (preferred) or contact the repository owner via GitHub (@imtmcdonald) with:

- A description of the issue and impact
- Steps to reproduce (or a proof-of-concept)
- Affected app(s) / paths
- Any relevant logs or screenshots (redact secrets)

## Scope

This repository contains multiple apps. Please include which app folder is affected (for example `chowsr/`).
